# Movie Library — Личная кинотека

**Автор:** Узденов Рамазан

## Описание

Приложение с графическим интерфейсом (GUI) для хранения информации о фильмах. Позволяет:
- Добавлять фильмы с названием, жанром, годом выпуска и рейтингом.
- Просматривать список всех фильмов в таблице.
- Фильтровать фильмы по жанру и году выпуска.
- Сохранять и загружать данные в формате JSON (автоматически).
- Проверять корректность ввода данных (год — число, рейтинг от 0 до 10).

## Требования

- Python 3.6+
- Стандартные библиотеки: `tkinter`, `json`, `os`

Никаких дополнительных установок не требуется.

## Запуск

```bash
python movie_library.py
